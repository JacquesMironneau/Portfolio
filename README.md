# Portfolio

Flask based portfolio to present my different projects.

### Set up and run:
```
python3 -m venv .
. venv/bin/activate
pip install -r requirements.txt
export FLASK_APP=view.py
flask run
```

To add a project:  
`/add/`  
To display the existing projects  
`/projects/`
